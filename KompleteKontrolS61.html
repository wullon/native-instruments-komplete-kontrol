NATIVE INSTRUMENTS – KOMPLETE KONTROL S 61

Important note: For the script to work, you need to load the Bitwig Studio preset into the Komplete Kontrol S using the Native Instruments Controller Editor.
TRANSPORT
PLAY Toggle between play and pause.
REC Toggle arrangement record on and off.
STOP Press short to stop playback, press again to move playhead to start.
STOP + LOOP Press and hold STOP, then press LOOP repeatedly to cycle through the modes.
LOOP Toggle arrangement loop on and off.
RWD Move the playhead backwards
FFW Move the playhead forwards
1. MIX MODE
E 1-8 Track volume.
STOP + RWD Move track bank up/left.
STOP + FFW Move track bank down/right.
2. TRACK MODE
E 1 Current track volume.
E 2 Current track pan.
E 3-8 Current track send 1-6.
STOP + RWD Move current track up/left.
STOP + FFW Move current track down/right.
3. DEVICE MODE
E 1-8 Current device macros.
STOP + RWD Previous device.
STOP + FFW Next device.
4. USER MODE
E 1-8 User controls freely mappable via right mouse-click "Learn Controller Assignment...".
STOP + RWD User bank 1.
STOP + FFW User bank 2.
